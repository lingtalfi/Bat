getCountryAlpha3CodeByAlpha2Code
-------------
2017-10-28



```php
str|mixed    getCountryAlpha3CodeByAlpha2Code (str:alpha2Code, mixed:default = "USA")
```

Return the ISO 3166-1 alpha-3 code from an ISO 3166-1 alpha-2 code.
