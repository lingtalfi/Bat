HttpTool
=====================
2017-06-27



This class contains functions for manipulating HttpProtocol.




post
-------------
2017-06-27



```php
str|false    post ( string:uri, array:data=[] )
```

Post the given data to the given uri, and return the result.




isHttps
-------------
2017-12-11



```php
bool    isHttps ()
```
